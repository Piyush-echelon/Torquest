# TorrHunt Capacitor/Volt Build Configuration

## 📁 Project Structure

```
android/
├── capacitor.config.json    # Capacitor configuration
├── volt.config.json         # Volt Build configuration
├── ionic.config.json        # Ionic/Capacitor config
├── package.json             # Dependencies
├── www/                     # Web assets (built)
│   ├── index.html
│   ├── styles.css
│   ├── app.js
│   └── icon.png
├── res/                     # Resources
│   ├── icon/               # App icons
│   └── screen/             # Splash screens
└── android/                 # Generated by Capacitor
    └── app/
        └── src/
```

## 🚀 Volt Build Instructions

### Step 1: Prepare Your Project

```bash
cd android

# Install dependencies
npm install

# Sync web assets with Capacitor
npx cap sync
```

### Step 2: Upload to Volt Build

1. **Go to Volt Build** (https://volt.build or your Volt provider)
2. **Create new project**
3. **Upload these files:**
   - `capacitor.config.json`
   - `volt.config.json`
   - `package.json`
   - `www/` folder (all contents)
   - `res/` folder (all contents)
   - `ionic.config.json`

### Step 3: Configure Build

In Volt dashboard:
- **Platform:** Android
- **App ID:** `com.torrhunt.app`
- **App Name:** `TorrHunt`
- **Version:** `1.0.0`
- **Min SDK:** 22
- **Target SDK:** 33

### Step 4: Build & Download

1. Click **Build**
2. Wait for build to complete (~3-5 minutes)
3. Download APK
4. Install on your Android device

## 🔧 Capacitor Commands

```bash
# Install dependencies
npm install

# Sync web assets
npx cap sync

# Add Android platform
npx cap add android

# Open in Android Studio
npx cap open android

# Build for production
npx cap build android

# Run on device
npx cap run android
```

## 📱 Required Permissions

The app requests these Android permissions:
- `INTERNET` - Search torrents
- `ACCESS_NETWORK_STATE` - Check connectivity
- `ACCESS_WIFI_STATE` - Network info

## ⚙️ Configuration Options

### Change App Name
Edit `capacitor.config.json`:
```json
{
  "appName": "Your App Name"
}
```

### Change App ID
Edit `capacitor.config.json`:
```json
{
  "appId": "com.yourcompany.app"
}
```

### Change Package Name
Edit `volt.config.json`:
```json
{
  "volt": {
    "appId": "com.yourcompany.app"
  }
}
```

## 🐛 Troubleshooting

### Build Fails
- Check all config files are valid JSON
- Ensure `www/` folder exists with all files
- Verify icon files are correct sizes

### App Crashes
- Check `capacitor.config.json` syntax
- Ensure minimum SDK is 22 or higher
- Verify all plugins are installed

### Volt Build Errors
- Upload all required config files
- Check file paths in volt.config.json
- Verify permissions are correct

## 📦 Build Output

After successful build, you'll get:
- `app-debug.apk` - For testing
- `app-release.apk` - For distribution (needs signing)

## 🔐 Signing for Release

For Google Play:
1. Create keystore file
2. Upload to Volt Build
3. Configure signing in volt.config.json
4. Build release APK
